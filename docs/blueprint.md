# **App Name**: AttendEase

## Core Features:

- Employee Management (CRUD): Create, read, update, and delete employee records stored in browser LocalStorage.
- Attendance Input: Allow employees to log their attendance (clock-in/clock-out) which is stored locally.
- Report Generation: Generate simple attendance reports based on the locally stored data, filtering by date and employee.
- Data Persistence: Use LocalStorage to persist employee and attendance data in the browser for an MVP implementation.

## Style Guidelines:

- Primary color: A calm blue (#5DADE2) to inspire confidence and reliability.
- Background color: Very light desaturated blue (#F0F8FF).
- Accent color: A subdued purple (#A98DFF) for interactive elements.
- Body and headline font: 'Inter', a versatile sans-serif font for clear readability.
- Use a set of minimalistic icons from a library like Heroicons or FontAwesome to represent actions and data.
- Employ a clean, grid-based layout using Tailwind CSS utilities to ensure responsive design and easy navigation.
- Subtle transitions and animations for user interactions to provide feedback and enhance user experience.
